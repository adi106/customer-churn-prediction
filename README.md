# Customer Churn Prediction Using Machine Learning

## Problem
Predict customer churn to help businesses reduce customer loss and improve retention strategies.

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Approach
- Data preprocessing and feature encoding
- Model training using Logistic Regression and Random Forest
- Model evaluation and comparison using standard metrics

## Outcome
The model identifies customers at high risk of churn, supporting data-driven business decisions.
